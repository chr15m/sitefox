csrf-token

(csrf-token)

Returns the CSRF token passed by Sitefox in the XSRF-TOKEN header. Pass this value when doing POST requests via ajax:

(js/fetch "/api/endpoint"
          #js {:method "POST"
               :headers #js {:Content-Type "application/json"
                             :XSRF-Token (csrf-token)}
                             :body (js/JSON.stringify (clj->js data))})
