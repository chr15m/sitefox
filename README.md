Set up a ClojureScript web server in one command using [sitefox](https://github.com/chr15m/sitefox) and [nbb](https://github.com/borkdude/nbb).

```shell
npm create sitefox-nbb mywebsite
cd mywebsite
npx nbb server.cljs
```
